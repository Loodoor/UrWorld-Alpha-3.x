Clavier :
'Up' : permet de monter
'Down' : permet de descendre
'Left' : permet d'aller à gauche
'Right' : permet d'aller à droite
'0' : ouvre l'invite pour le tchat
'1' : fullscreen -> fenêtré
'2' : passe ou non en mode créatif
'3' : change de shader
'4' : (en réseau) affiche ou non les
    joueurs
'5' : joue ou non la musique
'6' : parler à quelqu'un
'7' : choix de la taille du FOV
'9' : affiche l'interface de debug
'SHIFT' : ouvre l'inventaire
'SPACE' : permet de sauter
'ECHAP' : sauvegarder et revient au
    menu
'RETURN' : permet de courir ou non
En mode Gamer:
'Z' : permet de monter
'S' : permet de descendre
'Q' : permet d'aller à gauche
'D' : permet d'aller à droite
'E' : ouvre l'inventaire
'SPACE' : active ou non le mode
    testeur

Souris :
'molette' : permet de changer de
    bloc courant
'left clic' : pose un bloc
'right clic' utilise l'effet associé
    à un bloc si il y en a un
'molette clic' : sélectionne le bloc
    sous le pointeur
